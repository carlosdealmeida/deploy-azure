### ` 🔍 Sobre`

<p align="justify">Deploy de API ASP.NET CORE, EF Core e SQL Lite no Azure por meio da ferramenta GitHub Actions</p>

### ` 💡 Tópicos abordados`
* Azure
* SQL Lite
* Deploy automatizado
* CI/CD
* DevOps
* GitHub Actions

### `🔎 Como utilizar?`

<p>$ git clone https://github.com/devjamesbrandao/deploy-azure.git</p>

<p>$ dotnet restore</p>

<p>$ dotnet build</p>

<p>$ dotnet run</p>
